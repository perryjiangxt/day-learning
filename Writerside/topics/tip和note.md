# tip和note

## tip使用

```
> 这是md语法tip，但是注意不能换行。也可以在下边指定style,值可为tip、note、warning
> style默认值为tip
{style="note"}
```
### 效果如下

> 这是md语法tip，但是注意不能换行。style默认值为tip。
{style="tip"}


> 这是md语法tip，但是注意不能换行。style为note
{style="note"}


> 这是md语法tip，但是注意不能换行。style为warning
{style="warning"}
